# 🏥 MediPredict – Smart Healthcare Management System

A full-stack Hospital Management System built using Java, Spring Boot, JDBC, and MySQL to automate hospital operations like patient management, appointment scheduling, and billing.

---

## 🚀 Problem Statement

Traditional hospital systems rely on manual processes which lead to:
- Data redundancy and errors  
- Long waiting times  
- Inefficient billing  
- Poor record management  

---

## ✅ Solution

This system digitizes hospital workflows by:
- Managing patient and doctor records  
- Automating appointments  
- Generating bills  
- Providing REST APIs for smooth communication  

---

## ✨ Features

- 👨‍⚕️ Doctor Management  
- 🧑 Patient Registration  
- 📅 Appointment Booking  
- 💳 Billing System  
- 🚑 Ambulance Management  
- 🏥 Room Allocation  

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot, REST APIs, JWT, RBAC
- **Frpntend:** React.js
- **Database:** MySQL  
- **Connectivity:** JDBC  
- **API Testing:** Postman  

---

## 🧩 Architecture

- Follows **MVC architecture**
- REST APIs handle communication between frontend and backend  
- MySQL database stores structured hospital data
